zsGauge
directive in module zeus.viz

Description

A directive for rendering a gauge with a needle

Usage

as element:
<zs-gauge
       value="{number}"
       max-value="{number}"
       [back-color="{string}"]
       [fill-color="{string}"]>
</zs-gauge>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
valuenumber

Current value to be indicated by gauge

maxValuenumber

Max possible value

backColor
(optional)
string

The color of unfilled portion of the gauge

fillColor
(optional)
string

The color of filled portion of the gauge
